Over15 is a personal experiment of a virtual machine, an hypothetical and simple 16-bit microcontroller. It was part of a bigger project but now remains here :)

The microcontroller itself is an own design, as well as the assembly language and the bytecode that the microcontroller executes. The documentation is available here (it is a Markdown document, converted to HTML thanks to Showdown). Without it I couldn't implement anything, in fact it worked as my design notes.

In addition to the virtual machine, I developed three tools: the assembler, the debugger, and the disassembler. Both the assembler and the debugger have a fully functional front-end: you can write a program using the assembler tool, get its binaries (bytecode) in Base64, paste them into the debugger tool and watch program running. You can find a couple of assembly code examples into the assembly tool.
